Добавляет панель в редактор Gutenberg для экспорта содержимого любой записи или страницы в формат Markdown (.md файл или в буфер обмена).

== Описание ==

**Simple Export to Markdown** добавляет небольшую область внутри редактора блоков WordPress, гд в один клик вы можете экспортировать текущую запись или страницу в Markdown — либо скачать файл `.md`, либо скопировать текст прямо в буфер обмена.

Плагин использует библиотеку [Turndown](https://github.com/mixmark-io/turndown) для корректного преобразования блоков Gutenberg в синтаксис Markdown, сохраняя заголовки, абзацы, изображения, списки, цитаты и другие стандартные элементы.

== Возможности ==

* Экспорт любой записи или страницы Gutenberg в Markdown.
* Два режима: **Download .md** или **Copy to Clipboard**.
* Автоматически генерируемый YAML front matter (`title`, `slug`, `date`, `categories`, `tags`).
* Работает нативно внутри редактора блоков — без отдельных страниц настроек.
* Лёгкий (менее 14 KB JS), без трекинга, без внешних API-запросов.

== Требования ==

* WordPress 6.0 или новее.
* PHP 7.4 или новее.
* Активный редактор блоков Gutenberg.

== Установка ==

1. Загрузите папку плагина в каталог `/wp-content/plugins/` или установите плагин напрямую через экран плагинов WordPress.
2. Активируйте плагин через меню «Плагины».
3. Откройте любую запись или страницу в редакторе Gutenberg — справа появится новая панель **Экспорт в Markdown**.
4. Нажмите **Скачать .md**, чтобы сохранить файл Markdown, или **Копировать Markdown**, чтобы скопировать сгенерированный текст в буфер обмена.

== Частые вопросы (FAQ) ==

= Где находится кнопка экспорта? =
В боковой панели редактора Gutenberg (если панель скрыта — нажмите иконку «Настройки»). Панель называется «Export to Markdown».

= Работает ли плагин в Classic Editor? =
Нет. Плагин работает только в современном блочном редакторе (Gutenberg).

= Можно ли экспортировать кастомные блоки? =
Да. Кастомные блоки будут сериализованы в HTML перед конвертацией. Обычный HTML корректно отображается в итоговом Markdown.

= Сохраняет ли плагин файлы на сервере? =
Нет. Вся конвертация и скачивание выполняются локально в браузере — это безопасно и не затрагивает сервер.

== Скриншоты ==

1. Редактор Gutenberg с панелью **Экспорт в Markdown** в правой боковой колонке.
2. Пример Markdown-файла с автоматически созданным YAML front matter.
